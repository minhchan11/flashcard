JAVASCRIPT AND jQUERY FLASHCARD

JavaScript
JavaScript is a programming language commonly used in web development
Operators
A symbol that is used to perform an operation.
Variables
containers for storing data values.
Variable naming conventions
Variables with multiple words should always use an underscore between words.
Functions
A JavaScript function is a block of code designed to perform a particular task.
Methods
The actions that can be performed on objects.
Arguments
Function arguments are the real values passed to (and received by) the function.
Parameters
Function parameters are the names listed in the function definition.
Return
The return statement stops the execution of a function and returns a value from that function
Chaining methods
A technique that can be used to simplify code in scenarios that involve calling multiple functions on the same object consecutively
Strings
Groups of characters, like 'JavaScript', 'Hello world!'
Booleans
A JavaScript Boolean represents one of two values: true or false.
Undefined
The undefined property indicates that a variable has not been assigned a value.
NaN
The isNaN() function determines whether a value is an illegal number (Not-a-Number).
Escape
The escape() function encodes a string.
Alert
The alert() method displays an alert box with a specified message and an OK button.
Comments
JavaScript comments can be used to explain JavaScript code, and to make it more readable.JavaScript comments can also be used to prevent execution, when testing alternative code.
jQuery
A cross-platform JavaScript library designed to simplify the client-side scripting of HTML.
Attributes
Attributes provide additional information about HTML elements.
